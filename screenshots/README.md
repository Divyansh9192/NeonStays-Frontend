## Guidelines

- **Format**: PNG or JPG
- **Size**: 1920x1080 or 1440x900 pixels (recommended)
- **Naming**: Use descriptive, lowercase names with hyphens (e.g., `homepage.png`)
- **Quality**: Use high-quality screenshots that showcase the UI clearly
